# VDEX Trader

VDEX 거래소 자동 매매 프로그램

## 설치

```bash
pip install -r requirements.txt
```

## 실행

```bash
python vdex_gui.py
```

## 기능

- 🔐 지갑 서명 로그인
- 💰 실시간 잔고/포지션 조회
- 📈 마켓 주문 (매수/매도)
- 🔄 포지션 청산
- 🤖 자동 반복 거래

## 사용법

1. 지갑 주소와 개인키 입력
2. "Nonce 생성 & 서명" 클릭
3. "로그인" 클릭
4. 자동 거래 설정 후 "자동 거래 시작"
